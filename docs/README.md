# Índice de documentación

En esta carpeta  se encontraran toda la documentación relacionada con el proyecto, organizada para que se puedan ubicar fácilmente tanto la parte funcional como la técnica. La idea es que cualquier persona, ya sea de negocio, desarrollo o soporte, pueda entender el sistema según su necesidad.

## Documentos principales

- `arquitectura-c4-ejecutiva-explicacion-corta.md`
  - versión resumida de la arquitectura para negocio, revisión ejecutiva o presentación rápida
- `arquitectura-c4.md`
  - documentación C4 detallada con niveles, flujos, decisiones técnicas y riesgos
- `manual-de-usuario.md`
  - guía de uso del sistema pensada para usuarios finales
  - incluye login renovado, edición de usuarios con avatar y preparación de solicitud de compra antes del PDF
- `base-de-datos-prueba.md`
  - explica el DDL y DML entregados para crear y poblar la base de datos del proyecto
- `creditos-imagenes-catalogo.md`
  - referencia breve de la fuente usada para las imágenes del catálogo demo

## Archivos de apoyo en la raíz

- `README.md`
  - punto de entrada principal para ejecutar y entender el proyecto
- `CHANGELOG.md`
  - resumen de mejoras y entregables incorporados durante la solución
