# Créditos de imágenes del catálogo

Este documento se crea con el fin de dejar constancia de la fuente de las imágenes usadas en el catálogo demo del sistema.

## Alcance

Las imágenes fueron utilizadas para que el detalle de producto, la galería y los reportes reflejen artículos más cercanos a los productos reales del inventario.

No representan acuerdos comerciales ni marcas oficiales. Se usan únicamente con fines demostrativos dentro de esta solución técnica.

## Fuente principal

- Banco utilizado: `Pexels`
- Tipo de uso: imágenes gratuitas para demostración del catálogo

## Productos y referencia de imagen

- `Laptop Pro 14`
  - imágenes descargadas desde fotografías de laptop de trabajo en entorno de oficina
- `Monitor 27 IPS`
  - imágenes descargadas desde fotografías de monitores de escritorio y estaciones de trabajo
- `Mouse Ergonomico`
  - imágenes descargadas desde fotografías de mouse ergonómico o mouse de oficina
- `Teclado Mecanico`
  - imágenes descargadas desde fotografías de teclados mecánicos y escritorios de trabajo
- `Scanner Industrial`
  - imágenes descargadas desde fotografías de escáner industrial o escáner manual de inventario
- `Lector de Codigos`
  - imágenes descargadas desde fotografías de lector de código de barras o lector de punto de venta

## Ubicación en el proyecto

Las imágenes están almacenadas en:

- `Inventario.Web/wwwroot/images/products/catalog`

## Notas de uso

- Las imágenes fueron optimizadas para mejorar carga web y uso en PDFs.
- También se generaron imagebes  miniaturas para reportes.
- Si en el futuro se reemplazan por fotografías propias del negocio, se debe actualizar este documento.
